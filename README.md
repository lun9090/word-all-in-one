# word-all-in-one
Word办公助手
## 背景
工作中涉及到word比较多，一直在用小恐龙办公助手和快点，功能很庞大，但是我常用的功能不多，为了能够减少在编制文档过程中的操作链路，查阅网上信息，自行开发本插件，因开发时间比较久，前面版本比较多，并未正式发布，所以见谅。
